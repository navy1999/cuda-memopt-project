Frontend for **CUDA Matrix Multiplication Optimizations**: benchmark charts, autotune results, report link, and WebGPU + local-CUDA demos.

## Deploy on Vercel

1. Import the **repository** (not the `frontend` folder alone).
2. Set **Root Directory** to `frontend`.
3. Build and deploy. The build runs `node scripts/copy-data.js` to copy `../results` and `../report` into `public/`; ensure those folders exist in the repo (or the build will still succeed with empty data).

## Getting Started (local)

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
